Como usar 🚀

Esta página é uma ferramenta estática:

  • ❌ Não salvamos nada: Se der F5, as informações coladas somem.
  • ✅ Dica: Mantenha suas perguntas em um bloco de notas e cole aqui sempre que for estudar.
  • ✅ O que colar: O texto deve começar com [ e terminar com ].
  • ✅ Exemplo p/referência: [ { "p": "Quantos ossos tem um corpo adulto?", "r": "206 ossos" }, { "p": "Quanto é 9 x 9?", "r": "81" } ]
